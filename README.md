# RunUp 🏃

Aplicación Android nativa para el seguimiento y planificación de entrenamientos de running.

## Tecnologías
- Kotlin + Jetpack Compose
- Google Maps SDK
- Laravel (PHP 8.2) + MySQL
- Claude API (Anthropic)

## Funcionalidades
- Registro de entrenamientos geolocalizados en mapa interactivo
- Planificación de entrenamientos con IA
- Gestión de zapatillas deportivas
- Menús de alimentación personalizados con IA
- Panel de administración de usuarios
